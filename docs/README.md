# Documentation Assets

This folder contains static assets referenced by the main README.

## Banner image

Place your banner image here as **`banner.png`**.

### Recommended dimensions

- **Width:** 1280–1920 px
- **Height:** 400–640 px
- **Aspect ratio:** roughly 3:1 to 5:1 (wide banner format)
- **Format:** PNG with transparency, or solid background

### Reference in the README

The main `README.md` references the banner with:

```html
<p align="center">
  <img src="docs/banner.png" alt="Ouster ROS 2 Driver and CycloneDDS Setup" width="100%"/>
</p>
```

Once you place `banner.png` in this folder, the image will render at the top of the repository page on GitHub.
